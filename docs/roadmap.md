# Roadmap

## Fase 1: Plataforma base

- CLI y SQLite local.
- Pipeline de oportunidades.
- SEO pack y estructura de libro.
- Workflow n8n inicial.

## Fase 2: Datos reales

- Adaptador de tendencias.
- Adaptador de marketplace.
- Importador de reportes KDP.
- Dashboard de oportunidades.

## Fase 3: Produccion editorial

- Generacion asistida de manuscrito por capitulos.
- QA de estilo, originalidad y claims.
- Gestion de aprobaciones humanas.
- Versionado de assets.

## Fase 4: Escala

- PostgreSQL.
- Cola de jobs.
- Observabilidad.
- Multi-marca y multi-idioma.
- Experimentos de precio, ads y packaging.
